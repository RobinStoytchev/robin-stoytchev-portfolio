Coca-Cola Brand Colors

Exploring the Palette

This section visually presents the core and data-specific colors associated with the Coca-Cola brand identity, based on the provided guidelines. Each swatch displays the color and its corresponding RGB value.

The colors are defined using modern OKLCH values in the CSS for wider gamut and better perceptual uniformity, while displaying the standard RGB values for reference.

Core Colours

Deep Red/Brown RGB 36-0-0
White RGB 245-245-245
Core Red RGB 244-0-9
Light Red RGB 255-128-128
Cyan RGB 195-255-255
