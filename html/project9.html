Coca-Cola Brand Colors

This section visually presents the core and data-specific colors associated with the Coca-Cola brand identity, based on the provided guidelines. The colors are defined using modern OKLCH values in the CSS for wider gamut and better perceptual uniformity, while displaying the standard RGB values for reference on each card.

Core Colors

Deep Red/Brown

Core Color

RGB: 36, 0, 0

A rich, grounding color that conveys strength and stability. Perfect for foundational elements.

Pure White

Core Color

RGB: 245, 245, 245

A clean and crisp white, offering a sense of space, clarity, and modern simplicity.

Core Red

Core Color

RGB: 244, 0, 9

A vibrant and energetic red, commanding attention and signaling importance or passion.

Light Red

Core Color

RGB: 255, 128, 128

A softer, more approachable red, evoking warmth, and subtle emphasis.

Arctic Cyan

Core Color

RGB: 195, 255, 255

A bright and refreshing cyan, reminiscent of clear skies or cool waters. Ideal for highlights.

Data & Neutral Colors

Vibrant Teal

Data Color

RGB: 0, 230, 160

A dynamic teal that blends tranquility with invigorating energy, great for calls to action.

Rich Gold

Data Color

RGB: 255, 215, 0

A luxurious and optimistic gold, symbolizing quality, achievement, and warmth.

Deep Purple

Data Color

RGB: 70, 60, 200

A sophisticated and creative purple, often associated with wisdom, luxury, and ambition.

Playful Pink

Data Color

Light Gray

Neutral Color

Medium Gray

Neutral Color
